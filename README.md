# Flask Clipboard App

A simple Flask app with a full-screen text area and floating buttons.

## Quick Start

1. Clone the repository:
   ```sh
   git clone https://github.com/RishavT/seeplusplus
   cd your-repo-name
   ```

2. Install vercel:
   ```sh
   npm i -g vercel
   ```

3. Run the application:
   ```sh
   vercel dev
   ```
